# 🎓 Trenér převodů jednotek v2.0

Interaktivní PWA aplikace pro procvičování převodů fyzikálních jednotek.

![Verze](https://img.shields.io/badge/verze-2.0-green)
![Licence](https://img.shields.io/badge/licence-CC--BY--NC--SA--4.0-blue)

## ✨ Hlavní funkce

- **3 režimy:** Učení, Test, Hra
- **5 obtížností:** Start → Standard → Boss → God → God²
- **Páskový systém** (motivace jako v karate)
- **Google Forms integrace** pro učitele
- **Offline PWA** - funguje bez internetu
- **Tmavý režim** + 3 fonty pro dyslektiky
- **Single-file HTML** - celá aplikace v jednom souboru

## 🚀 Jak začít

### Pro žáky:
1. Stáhni `Unit_Converter_v2.0.html`
2. Otevři v prohlížeči (Chrome, Firefox, Safari)
3. Hraj a uč se!

### Pro učitele:
1. Přečti [Příručku pro učitele](docs/Prirucka_pro_ucitele.pdf)
2. Zkopíruj si Google Forms šablonu (návod v příručce)
3. Sdílej HTML s žáky
